# 📊 Dashboards Power BI — DataSimple

Ejemplos de dashboards construidos para contextos reales de la región.
Los archivos `.pbix` se pueden descargar y explorar libremente.

> Los datos son simulados pero basados en casos reales de comercios y empresas de Santa Fe.

---

## Dashboards disponibles

### 1. Ventas Mensual — Comercio Minorista
`/ventas-mensual/`

Panel de control con:
- Ventas por día, semana y mes
- Productos más vendidos
- Comparativo con mes anterior

**Vista previa:**
![preview](ventas-mensual/preview.png)

---

### 2. Stock — Comercio o Distribuidora
`/stock-comercio/`

Panel para seguimiento de inventario:
- Stock actual por producto
- Alertas de stock mínimo
- Rotación de productos

**Vista previa:**
![preview](stock-comercio/preview.png)

---

### 3. Rentabilidad Agro — Campaña Gruesa
`/rentabilidad-agro/`

Dashboard orientado a productores o acopios:
- Costo vs ingreso por lote o cultivo
- Evolución de precios (cargados manualmente)
- Margen estimado por campaña

**Vista previa:**
![preview](rentabilidad-agro/preview.png)

## ¿Cómo usar estos archivos?

1. Descargá el archivo `.pbix` de la carpeta correspondiente
2. Abrilo con [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (gratuito)
3. Explorá el modelo, las medidas y las visualizaciones
4. Si querés conectarlo a tus propios datos, contactanos
